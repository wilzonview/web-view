# Duber Website Wireframe

Responsive HTML/CSS wireframe for Duber, an Audio Experience Studio positioned around immersive atmosphere for hospitality venues.

## Description

This wireframe translates the Duber strategy direction into a presentable website and future webapp structure. It uses the `Sapphire Future + Human Atmosphere` direction: an obsidian and midnight-sapphire interface system with electric-blue CTAs, subtle glass panels, restrained motion cues, and warm hospitality atmosphere moments.

The prototype is designed to guide the first public website build for Duber. It emphasizes atmosphere, memory, guest experience, and venue value rather than DJ, AV rental, or equipment-first messaging.
