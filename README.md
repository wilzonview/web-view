# Web View – Roofing Landing Page

This repository contains a deploy-ready static landing page for a roofing sales framework.  
It is used to present website concepts to clients before final hosting and deployment decisions are made.

## Tech stack

- HTML for structure
- CSS for layout and styling
- JavaScript for basic interactivity
- Assets organized under `css/`, `js/`, and `images/`

## Usage

You can open `index.html` directly in a browser or serve the site with any static file server.

## Live preview

View the live site here: https://wilzonview.github.io/web-view/
